Проект направлен на создание системы автоматического распознавания речи (Speech-to-Text) для северномансийского языка - одного из языков России, относящегося к обско-угорской группе уральской языковой семьи и находящегося под угрозой исчезновения.
Цель проекта - разработать и сравнить подходы к созданию Speech-to-Text анализатора для северномансийского языка, что позволит:
- Ускорить транскрибирование аудиоматериалов
- Облегчить работу лингвистов и исследователей
- Способствовать сохранению и цифровизации языкового наследия
  
Основой проекта служит архив газеты «Лӯима Сэ̄рипос», из которого были взяты озвученные своими авторами  новостные тексты на северномансийском языке (сосьвинский и верхнелозьвинский диалекты). Каждый текст содержит 50–110 предложений. Аудиозаписи размечены на предложения в программе ELAN.

Metadata_verbessert.csv - метаданные всех аудиозаписей. Отсутствует разбиение на подвыборки, кириллическая запись. Размечены по носителям, длинам, содержание записей.

khanty_whisper.csv - результаты эксперимента с Zero-Shot с Whisper, обученной для хантыйского языка.

my_dataset_test.csv - метаданные, размеченные в кириллице и латинице. Результат разбиения на подвыборки - тестовая подвыборка.

my_dataset_train.csv -  метаданные, размеченные в кириллице и латинице. Результат разбиения на подвыборки - тренировочная подвыборка.

my_dataset_val.csv -  метаданные, размеченные в кириллице и латинице. Результат разбиения на подвыборки - валидационная подвыборка.

whisper_base.csv - результаты эксперимента Zero-Shot с моделью WhisperBase

Промежуточные результаты.pdf - результаты первых экспериментов с моделью Whisper с применением методов Zero-Shot и DFT.

Руководство по разметке-2.docx - конвенции разметки аудиозаписей.

Сравнение подходов к созданию Speech-to-Text анализатора для северномансийского языка-2.pdf - подорбное описание проекта.
